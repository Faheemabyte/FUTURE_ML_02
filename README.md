Support Ticket Classification System

Future Interns Machine Learning Internship - Task 2

 Overview
 
A Machine Learning system that automatically classifies customer support tickets 
into categories and assigns priority levels using Natural Language Processing (NLP).

Features

- Automatic ticket category classification (Technical, Billing, Refund, Shipping)
- Priority level assignment (High, Medium, Low)
- Text preprocessing and cleaning
- TF-IDF vectorization
- Support Vector Machine (SVM) model
- Confusion matrix visualization

Tools and Libraries Used

- Python
- Pandas
- Scikit-learn
- NLTK
- Seaborn
- Matplotlib
- Joblib

Model Performance
- Overall Accuracy: 70%+
- Best performing category: Technical
- Model: Support Vector Machine (SVM)

Files
- support_ticket_classification.ipynb - Main notebook
- ticket_classifier_model.pkl - Saved trained model
- tfidf_vectorizer.pkl - Saved TF-IDF vectorizer
